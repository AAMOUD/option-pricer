# Option Pricing Library

## Goal
Implement core option pricing models (Black-Scholes, Binomial/Trinomial Trees, Monte Carlo with variance reduction) in Python, later C++.

## Structure
- `quantlib_core/` - pricing engines
- `notebooks/` - demos & analysis
- `tests/` - unit tests
- `cpp/` - C++ skeleton

## Instructions
Activate the Python environment and run notebooks.
